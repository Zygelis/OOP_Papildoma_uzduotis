# OOP_Papildoma_uzduotis

## Aprasymas
Programa, kuri iš tekstinio failo išrenka:
- URL'us
- žodžius pasikartojančius daugiau nei 1 kartą
- eilutes, kuriose žodžiai pasikartoja daugiau nei 1 kartą

## Paleidimas
- Nusiclonuokite repositorija
- terminale pereikite į projekto katalogą
- paleiskite programą g++ main.cpp -o main ir ./main

## Failai
- main.cpp - pagrindinis programos failas
- input - aplankas laikantis savyje skirtingus teksto failus, kurie gali būti naudojami programoje. Taip pat šis aplankas laiko url_pabaigos.txt, kuris turi daugybe URL pabaigų.
- output - aplankas, kuriame bus laikomi išvesties failai, kai paleisite programą
